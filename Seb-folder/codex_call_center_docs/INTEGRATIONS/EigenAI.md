# Eigen AI Integration

This file covers two possible interpretations:
- **EigenPal** for document workflow automation
- **Eigent** for multi-agent orchestration

## Preferred role for this product
### EigenPal = document automation layer
Use EigenPal for:
- post-call document upload handling
- claims forms
- invoices
- identity/KYC documents
- contracts
- receipts
- supporting evidence extraction

## Secondary option
### Eigent = orchestration layer
Use Eigent if the product later needs:
- multiple background agents
- parallel checks
- tool-heavy back-office workflows
- human-in-the-loop supervision
